== Description ==

Hashtags for WordPress, BuddyPress and bbPress. Adds hashtag links to WordPress posts,BuddyPress activity and bbPress topics. Hashtags turn into links that are used to search items based on topics.

demo: http://demo.taptappress.com

== Installation ==

1. Upload plugin folder '/hashbuddy/' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Type a hashtag into activity stream. Click hastags to sort stream.
